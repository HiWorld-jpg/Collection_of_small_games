##编译环境
1. 把代码下载到本地，然后使用visual studio 2019（或者更高版本），打开项目目录下的 “小游戏合集.sln 这个文件”
2. 如果直接编译报错，进入visual studio 2019，右键这个项目，进入项目属性页，依次进入 “配置属性-高级-字符集” ，选择 “使用多字节字符集”。即可解决报错问题
